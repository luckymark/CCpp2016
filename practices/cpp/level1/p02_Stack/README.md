### 题目：IntStack（堆栈）

### 功能要求（一）：

实现一个IntStack类（后进先出FILO），可容纳100个int数据，并实现下面的操作：

1. push（入栈）
2. pop（出栈）
3. 判断是否满
4. 判断是否空

### 功能要求（二）：

实例化堆栈对象的时候，可指定该堆栈的容量

### 功能要求（三）（学了模板以后再做）：

将IntStack改造为类型无关的Stack

###流程
1.	搭建框架
2.	用数组实现固定大小的stack
3.	用数组指针实现可变大小的stack
4.	用链表实现可变大小的stack
>1.	实现并完善构造函数，append与pop方法并用for循环测试通过
>2.	完成空与满的两个判断函数，并更改测试代码用while进行测试通过
